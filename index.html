1. 오늘 하고 싶은 것 — 핵심 방향 4가지
① 콘텐츠 시스템 구축
  • 피드 + 스토리 분리 운영
  • 스토리 요일별 자동화 구조 설계
  • 피드 1개 → 스토리 5개 파생 구조
② 구매로 이어지는 콘텐츠 설계
  • 소재 언어 → 육아 상황어로 번역
  • 저장 → 링크 클릭 → 구매 퍼널 완성
  • 공감으로 멈추게 하고, 정보로 저장하게 하고,
    희소성으로 클릭하게 하고, 신뢰로 결제하게 한다
③ 크리에이터 생태계 확장
  • 로덬메이트 → 크리에이터로 성장 (영상 기획 같이 도와주기)
  • 스페셜 크리에이터: 제안 자유도 부여
  • 플랫폼 시딩: 키디 / 무신사 / 모이몰른
④ 분업 구조 확립
  • 선애님 → 주제 · 메시지 · 카피 · 검수
  • 웹디 → 비주얼 · 영상 실행 (사진·파일 선택 포함)
  • 자동화 → Meta 비즈니스 스위트 예약 발행
